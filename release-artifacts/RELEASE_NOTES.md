# My MD Viewer 0.1.2

탐색기에서 문서를 여는 흐름과 창 조작을 개선한 릴리스입니다.

## 변경 사항

- 탐색기에서 Markdown 파일을 더블클릭했을 때 앱 창을 복원하고 다른 창 앞으로 활성화
- 이미 실행 중인 앱으로 새 파일이 전달될 때도 해당 문서 창을 전면에 표시
- `Ctrl+W`로 현재 문서 창 닫기
- 도구 모음의 버전 표시를 제거하고 제목 표시줄에 `My MD Viewer - v0.1.2` 형식으로 표시

## 설치 참고

Windows Authenticode 코드 서명이 없는 개인용 빌드입니다. Windows SmartScreen 경고가 나타나면 다운로드 주소가 이 공개 저장소인지와 `SHA256SUMS.txt`의 파일 해시를 확인하세요.
