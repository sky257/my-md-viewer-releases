# My MD Viewer 0.1.1

읽기 흐름을 방해하지 않는 자동 업데이트와 확대 가능한 목차를 추가한 릴리스입니다.

## 변경 사항

- 실행 후 자동으로 업데이트를 확인하고 서명된 설치 파일을 백그라운드에서 다운로드
- 모든 문서 창을 닫을 때 준비된 업데이트를 자동 설치
- 여러 창에서도 업데이트를 한 번만 다운로드하도록 프로세스 단위로 통합
- 본문 확대율에 맞춰 목차의 글자, 간격, 아이콘과 너비를 함께 확대·축소
- 도구 모음에 현재 앱 버전 표시
- 공개 배포 저장소 전용 키를 사용하는 최소 권한 자동 릴리스 구성

## 설치 참고

Windows Authenticode 코드 서명이 없는 개인용 빌드입니다. Windows SmartScreen 경고가 나타나면 다운로드 주소가 이 공개 저장소인지와 `SHA256SUMS.txt`의 파일 해시를 확인하세요.
