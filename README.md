# My MD Viewer 릴리스

Windows 11에서 Markdown 문서를 빠르고 안전하게 읽기 위한 개인용 뷰어의 공식 바이너리 배포 저장소입니다.

- `My.MD.Viewer_*_x64-setup.exe`: Windows 11 x64 설치 파일
- `My.MD.Viewer_*_x64-portable.exe`: 무설치 실행 파일
- `latest.json`: 앱 내 업데이트 정보
- `SHA256SUMS.txt`: 배포 파일 무결성 확인용 SHA-256 해시

소스 코드는 비공개 저장소에서 관리하며 이 저장소에는 공개 배포 파일만 게시합니다. Windows Authenticode 코드 서명을 사용하지 않으므로 SmartScreen 경고가 표시될 수 있습니다.
